# FitTimer — Privacy Policy

This repository hosts the privacy policy for the **FitTimer** Android app,
served via [GitHub Pages](https://pages.github.com/).

**Live URL:** <https://j-aldama.github.io/fittimer-privacy/>

---

## What is FitTimer?

FitTimer is a training timer app for Android. It lets you run preset and
custom workout routines with series, repetitions, background execution,
sound cues and vibration.

- **Play Store:** _(pending publication)_
- **Source code:** private repository
- **Developer:** Juan Ignacio Aldama — [@j-aldama](https://github.com/j-aldama)

---

## About this repository

This repo contains a **single file** — `index.html` — with the full
privacy policy in Spanish (es-MX). GitHub Pages serves it from the `main`
branch, root folder.

### Why a separate public repo?

The FitTimer source code is kept private. Google Play and AdMob require
a publicly reachable URL for the privacy policy, so this repo exists
solely to host that HTML file without exposing the app's source.

---

## Updating the policy

1. Edit `index.html` (keep the "Última actualización" date current)
2. Commit and push to `main`
3. GitHub Pages redeploys automatically within 1–2 minutes
4. Hard-refresh the live URL to see the update (<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>R</kbd>)

## Scope of the policy

The document covers:

1. Who the responsible party is
2. What data the app does **not** collect
3. Data stored locally on the device (routines, settings, timer state)
4. Data collected by third parties (Google AdMob)
5. Android permissions requested
6. Children's policy (COPPA)
7. User rights
8. Security practices
9. How updates to the policy are announced
10. Applicable law (MX, plus notes for GDPR / UK-GDPR / CCPA)
11. Contact
